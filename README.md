# 短網址產生器

## 介紹
AC 2-3 A11作業，可將任意網址縮短。


## 功能 
- 可將輸入網址轉換為短網址
- 可透過短網址連結到原本的網址


## 開啟專案

1. 先下載專案至本地
2. 安裝node.js及npm
3. 透過終端機進入資料夾並輸入：

  ```bash
  npm run dev
  ```

4. 若看到此訊息則代表順利運行，打開瀏覽器進入到以下網址
  
  ```bash
  Listening on http://localhost:3000
  ```
